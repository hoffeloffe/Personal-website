# Roadmap

## Phase 1 (Current)

- Azure Static Web Apps in `westeurope`
- Separate `dev` and `prod` environments
- Bicep infrastructure in `infra/`
- GitHub Actions CI + infra + environment deploy workflows

## Phase 2 (Planned)

- Add basic observability (Application Insights and dashboards)
- Add smoke tests with a lightweight frontend test stack
- Improve release traceability (release notes and deployment records)

## Phase 3 (Candidate)

- Optional backend service to demonstrate broader Azure platform usage
- Preview environment strategy for pull requests
- Custom domain and DNS documentation hardening
