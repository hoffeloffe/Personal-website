# Architecture

## Phase 1 Scope

This repository hosts a frontend portfolio in `app/` and deploys it to Azure Static Web Apps (SWA).

- Region: `westeurope`
- Environments: `dev`, `prod`
- IaC: Bicep (`infra/`)
- CI/CD: GitHub Actions (`.github/workflows/`)

## Environment Topology

### Dev

- Resource Group: `rg-portfolio-dev-we`
- Static Web App: `swa-portfolio-dev-we`

### Prod

- Resource Group: `rg-portfolio-prod-we`
- Static Web App: `swa-portfolio-prod-we`

## Design Notes

- The app is a static React + TypeScript build generated by Webpack.
- Runtime behavior is controlled by `staticwebapp.config.json`.
- SPA routing falls back to `index.html`.
- Security headers are applied globally using SWA configuration.

## Out of Scope (Phase 1)

- Backend API
- Observability stack (App Insights / Log Analytics)
- Key Vault, Container Apps, AKS, Front Door, Functions
